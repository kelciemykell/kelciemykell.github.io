Target Audience

  1. Male/female, "outdoorsy", 20s-40s, enjoy hiking and spending time in nature
    Scenario:
    • "I want to go on a hike this saturday with some friends, what will the weather be like?"
    • "I am planning a camping trip and I want to know what the weather will be like this weekend?"
    • "what will the weatehr be like at Yosemite next week? I want to go there, but not when it is raining"
  2. Male, construction site manager, 30s-50s
    Scenario:
    • "we need to put the roof on the building this week, when is it supposed to rain next so we can get it done before then?"
    • " The client want the sidewalk poured this week. Which days will be the sunniest and warmest so the sidewalk will dry faster? "
